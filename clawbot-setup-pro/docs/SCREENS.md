# ClawBot Setup Pro — Screens (MVP)

## Mobile
1. **Welcome / Sign in**
   - Email, magic link
2. **Home**
   - Primary CTA: Start Setup
   - Device list (0..n)
3. **Purchase / Unlock**
   - Explains external checkout
   - Button: Continue to Checkout
   - Post-checkout: Refresh status
4. **Add Device (Pairing)**
   - QR + pairing code
   - Instructions: download desktop helper
5. **Select Template**
   - Basic local assistant / Home automation / etc.
6. **Setup Progress**
   - Step list, current step
   - Live logs (collapsible)
   - Retry / Pause / Stop
7. **Setup Complete**
   - Status: Running
   - Buttons: View Logs, Run Health Check, Stop/Start
8. **Troubleshooting**
   - Detected issue summary
   - Auto-fix attempted / recommended actions
   - Button: Request Expert Help
9. **Support**
   - Upload diagnostics
   - Message field

## Desktop Helper
1. **First Run / Pairing**
   - Enter pairing code
   - Approve device linking
2. **Setup Runner**
   - Shows current action + logs
   - Allows local cancel
3. **Diagnostics Viewer**
   - What will be shared
   - Redaction notes
